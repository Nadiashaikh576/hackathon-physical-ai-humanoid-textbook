## Image Descriptions

- `humanoid-robot.jpg` - Visual representation of a humanoid robot
- `ros2-architecture.png` - Diagram showing ROS 2 architecture for humanoid robots
- `isaac-sim-interface.png` - Screenshot of NVIDIA Isaac Sim interface
- `vla-pipeline.png` - Visualization of the Vision-Language-Action pipeline
- `digital-twin-simulation.png` - Example of digital twin simulation environment
- `voice-to-action-pipeline.png` - Diagram of voice-to-action conversion process
- `cognitive-planning-process.png` - Visualization of cognitive planning with LLMs
- `perception-to-manipulation-pipeline.png` - Complete perception-to-manipulation pipeline
- `complete-autonomous-system.png` - Overview of the complete autonomous humanoid system

These images are referenced in the documentation to enhance understanding of complex concepts.
