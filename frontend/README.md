# AI-Native Book: Physical AI & Humanoid Robotics

This documentation site teaches ROS 2 as the core middleware connecting AI intelligence to humanoid robot bodies. The content is designed for AI/software students new to robotics with basic Python knowledge.

## Modules

1. **Module 1: The Robotic Nervous System (ROS 2)**
   - Chapter 1: ROS 2 Fundamentals
   - Chapter 2: ROS 2 Communication
   - Chapter 3: AI-to-Robot Bridge

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm start
```

## Build

```bash
# Build the static site
npm run build
```

This site is designed to be deployed to GitHub Pages.
