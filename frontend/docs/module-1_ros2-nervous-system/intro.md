## Glossary

- **ROS 2**: Robot Operating System 2, a flexible framework for writing robotic software providing tools, libraries, and conventions.
- **Node**: A process that performs computation in ROS; nodes are organized into a graph structure for distributed computation.
- **Topic**: A named bus over which nodes exchange messages in a publish-subscribe pattern.
- **Service**: A synchronous request/response communication pattern between nodes.
- **Action**: An asynchronous communication pattern for long-running tasks with feedback.
- **rclpy**: The Python client library for ROS 2, providing Python developers access to all ROS 2 capabilities.
